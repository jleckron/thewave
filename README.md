# thewave

Find new friends while driving


Built on React Native and Firebase

To take changes:
```
  git pull origin master
  cd ios && pod install
 ```
To launch:
```
  cd thewave
  react-native run-ios
```


### Current State
- User Signup / Login
- User Password Recovery
- Add / Edit / Delete Vehicle Data
